# Retrieving a token from Auth0

```
curl --location --request POST "https://melvin-vermeer.eu.auth0.com/oauth/token" --header "Content-Type: application/x-www-form-urlencoded" --data-urlencode "client_id=8dvVsK8DJazXU3Lr8EBRiz4aOcjFuyho" --data-urlencode "username=***" --data-urlencode "password=***" --data-urlencode "grant_type=password" --data-urlencode "scope=openid"
```
